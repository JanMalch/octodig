About octodig

Version {{ version }}

octodig is a simple web app that allows you to browse any GitHub repository in an editor look. It supports syntax highlighting for over 150 languages, including code folding and more!

octodig is not an editor and you cannot commit files through it. Support for advanced features like resolving imports are out of scope.
The goal is to provide you a simple app, to open multiple files at once and keep track of them, without opening a dozen browser tabs.

For bonus convenience add this bookmarklet to open the GitHub repository, that you are currently on, here on octodig! 

javascript:window.open("https://octodig.web.app/open?url=" + location.pathname, "_blank");

How it’s build

octodig is completely open source and you can check out the full source code. Third-party licenses are available. To find out more you can read the project's documentation.

Frontend

The app is built with Angular 9 and uses @angular/material with materialdesignicons.com for the UI.

The code highlighting is done with Codemirror and rendering markdown is done with marked.

Authentication

You can login to have more GitHub requests per hour. This requires an auth provider, which is Google’s Firebase as it also hosts the website. octodig doesn’t store any user data in the database. By clicking the button on the home page you can remove the account and revoke the access for octodig at any time.
