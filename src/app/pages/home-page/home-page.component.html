Welcome to octdig

This project is no longer maintained. Consider using GitHub's Codespaces instead.

octodig is a simple web app that allows you to browse any GitHub repository in an editor look. It supports syntax highlighting for over 150 languages, including code folding and more!

octodig is not an editor and you cannot commit files through it. Support for advanced features like resolving imports are out of scope.
